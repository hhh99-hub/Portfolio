Project Details

The Online Examination Software project is a web-based application that enables teachers to conduct secure online exams. The frontend is developed using HTML, CSS, and JavaScript, while the backend utilizes Python with the Django framework. A notable feature of this software is the implementation of Amazon Rekognition, an AI-powered image and video analysis service, to ensure that only authorized students can participate in the exam. The software allows teachers to upload question papers, enables students to take exams online, and provides a grading system for teachers to evaluate and assign marks.
